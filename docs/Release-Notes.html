Release Notes

  • Latest release

2024/25 downstream fixes for Quickemu​

Available via qqX's freebird or in the qqX code itself

  • The July 2025 release features a huge new upgrade for QuickGet > fixes, sorts and adds over 25 distros

  • The use of floatversion allows multiple distro downloads not available in Quickemu

Whilst July 2024

  • Fixes problems causing Intel processors to run MacOS very slowly

  • Fixes the problem where telescoped/nested VM grind to a halt

  • Windows language fixes mean that all the downloads work, not just English

And March/Nov 2025

  • Fixes and then refines the Quickemu Windows Installer issue #1475

Creative differences​

There are always creative differences within projects.

The Quickemu project is no exception and the growing count of developers that walked away during 2024 should not affect you as a user.

Relationship with Quickemu​

More independence means that you get a faster moving program that is not being held back.

Meanwhile, qqX will continue to support virtual machines that have been created using QuickEmu or QuickGUI.

There used to be a qqx listing on the Quickemu wiki pages but things need to move on and need to evolve.
