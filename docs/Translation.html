Help with Translation

As with much of computing, users will benefit from some basic understanding of English

However:

The left hand SIDE MENU / DROPDOWN HAS READY COMPILED PAGES FOR PARALLEL TRANSLATION

Click here to Google translate THIS page

Built in Auto Translation​

These webpages use a React sub-system. The contents lists, on the right, have problems with auto-translation >>

Where available, use these:

Top - - Built-in-auto-translation - - Web Pages - - Settings - - Copy and Paste - - Chrome

 [tx] traduire de l'anglais   [set] open settings file in text editor  [conf] view or edit .conf file

The use of popup menu options is being expanded throughout the qqX 1.13 series releases.

An English or Latin Character keyboard, or keyboard overlay, may be required.

The tx command will now open the SIDE MENU parallel translations in the set default browser in over 51 languages

Some entries may occasionally refer to new features. These are either being added to the development branch, or may be updates to older implementations that sometimes show in a newer or slightly different format.
