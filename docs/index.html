qqX

quickemu quickget X terminal project​

SPIN-OFF, NEXT LEVEL FRONT-END FOR THE POPULAR QUICKEMU SYSTEM

  • Adds fixes, tools and general improvements to both Quickemu and Quickget

  • Huge, easy build range of installation-ready distros and desktops

  • Allows the easy running of Linux, Windows, MacOS, etc with QEMU / KVM

CAREFULLY WRITTEN AND CAREFULLY CHECKED

After lots of headache with qemu, quickemu, virtmanager etc, I finally found this which worked for me. Thank you. @YehiaEhab17 - issue #11

Any qqX built machines will still run using Quickemu and you can always switch or go back to QuickGUI

Simply Better​

  • Rapid development, simplified interface system, for easy extra features

  • Less CPU constrained than Docker/Pod systems

  • More up-to-date and much easier to mod than Gnome Boxes or VirtualBox

  • Much simpler than the Qemu front-ends based on LibVirt

Rolling release​

qqX makes lists of on-going Quickemu and QuickGui issues simply disappear

  • Choice of 3 easy-swap internal rolling release models: Leading Edge, Standard or Slow

qqX's built-in code system ensures that fixes and ideas can be quickly put to work

See the DROPDOWN menu for installation notes and full details on ALL sections

Ease of the GUI​

  • Download, right click and install

Power and simplicity of the command line​

  • Search the quickget distro list with simple alphabetical zooming

  • Filter distros by type: gaming, ultra-light, server ....

  • Includes distro homepage browser, link checker and download size reportage

Straightforward and easy to work with​

  • Choice of Custom Menus or 5 built-in menu styles

  • Distro storage size analysis and size disparity checks

  • And extended stop, start and resume for downloads

Built in Auto Translation​

 [tx] traduire de l'anglais   [set] open settings file in text editor  [conf] view or edit .conf file

Interface system allows parallel auto-translations in over 51 languages

Easy utilities​

  • Safeguard your machines with easy snapshot management

  • Make sure drives are healthy with Qcow2 repair, resize and diagnostics

  • Edit machine partitions by booting from maintenance ISO's

  • View inside virtual machines using a file manager

  • Adjust configurations with the tune-up wizard

  • And more

How to Install​

For standard Linux, most users will only need to install qemu spice-gtk and qqX

  • Quickemu and Quickgui may also be installed but are not essential.

That's it.

Reliability​

  • Fully compatible with existing Quickemu machines

  • Any qqX built machines will still run using Quickemu and you can always switch or go back to QuickGUI

  • All scripts are carefully Shellcheck linted and have full error handling routines.

  • Built-in updater included
